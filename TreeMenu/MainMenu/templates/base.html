Задача:

Нужно сделать django app, который будет реализовывать древовидное меню, соблюдая следующие условия:

  1. Меню реализовано через шаблонный тег.
  2. Все, что над выделенным пунктом - развернуто. Первый уровень вложенности под выделенным пунктом тоже развернут.
  3. Информация о меню хранится в базе данных.
  4. Редактирование меню осуществляется в стандартной административной панели Django.
  5. Активный пункт меню определяется исходя из URL текущей страницы.
  6. Меню на одной странице может быть несколько. Они определяются по названию.
  7. При клике на пункт меню происходит переход по заданному в нем URL. URL может быть задан как явно, так и через именованный URL.
  8. На отрисовку каждого меню требуется ровно 1 запрос к базе данных.

Нужен django app, который позволяет добавлять меню в базу данных через административную панель, и рисовать меню на любой нужной странице по названию.

Пример использования шаблонного тега для отображения меню: draw_menu 'main_menu'

При выполнении задания следует использовать только Django и стандартную библиотеку Python.

При решении задачи у вас не должно возникнуть вопросов. Если возникают вопросы, вероятнее всего, у вас недостаточно знаний.
